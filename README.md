# NoHotbarItemName

## Information

This Mod Removes the text of the name of the item your holding that shows above the hotbar.

This works by using a mixin that injects in the gui rendering and basically just calls to cancel the render.

You can also bind a key to disable / enable this mod/feature by setting a bind in 
"key.categories.nohotbaritemname.keybinds" for toggle hotbar item name.
